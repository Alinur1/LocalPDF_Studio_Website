System Requirements

Check compatibility with your device to ensure LocalPDF Studio runs smoothly

Windows

  • Operating System: Windows 10 (Version 22H2) or later.
  • Architecture: 64-bit (x64) only.
  • Processor: Intel or AMD CPU.
Note: Windows 7 and 8.1 are not officially supported due to Electron/Chromium limitations.

Linux - Snap Package

  • Compatibility: Works on all major distributions supporting snapd (Ubuntu, Fedora, Debian, Mint, Manjaro, etc.).
  • Architecture: 64-bit (x64) only.
  • Processor: Intel or AMD CPU.

Linux - AppImage (Portable)

  • Minimum Base: Ubuntu 20.04 (or equivalent).
  • Distro Compatibility:
    • Debian: 11 (Bullseye) or later.
    • Fedora: 32 or later.
    • Linux Mint: 20 or later.
    • Arch Linux: Most recent versions (Rolling release).
  • Dependencies: Requires Python 3.8.10 or higher (pre-installed on most modern distros).
  • Architecture: 64-bit (x64) only.

macOS

  • Operating System: macOS 13 (Ventura) or later.
  • Architecture: Intel (x64) Native.
  • Apple Silicon (M1/M2/M3/M4): Supported via Rosetta 2 translation.
Note: Native ARM64 support is currently not included; please ensure Rosetta is installed.

General requirements

  • Third party software: Ghostscript is needed to be installed. Tutorial: https://youtu.be/fKrnSytg_z4
  • RAM: 500MB minimum (2GB+ recommended for large PDF processing).
  • Disk Space: ~700MB for installation + additional space for temporary file processing.

Security & "Unsigned" Software Notice

LocalPDF Studio is 100% open-source and private. However, because I am an independent developer and not a massive corporation, this software is currently not digitally signed by Microsoft or Apple.

Signing certificates cost hundreds of dollars every year—money that currently comes out of my own pocket. Until the project is fully funded, you will see security warnings during installation.

How to Install (Manual Bypass)

For Windows Users

  1. Run the installer.
  2. When Windows Defender SmartScreen pops up, click "More info".
  3. Click "Run anyway".

For macOS Users

  1. Install the .dmg and drag the app to your Applications folder.
  2. Go to System Settings → Privacy & Security.
  3. Scroll down to the Security section.
  4. You will see: "LocalPDF Studio was blocked because it is not from an identified developer."
  5. Click "Open Anyway" and confirm with your password.
